# HOW TO RUN

---

## Clone về

1. Mở thư mục muốn tải về trong  `file experlorer` (bấm ctrl + e)
2. chuột phải ở thư mục muốn chọn để tải. Ví dụ tải vô thư mục DO_AN_4 thì chuột phải và DO_AN_4
    1. Bấm vào mở `terminal` hoặc `powershell` điều được
    2. gõ lệnh `git clone https://github.com/huynhlalyla/DO_AN_4.git`

## Chạy Web 

1. Mở server trên terminal
- Chuột phải dô thư mục `server` -> chọn cái nào có `open in terminal` (kiểu vậy)
- Trong terminal (nếu thấy nằm đúng thư mục server rồi) 
    1. Gõ `npm i` để cài thư viện - nếu cài rồi thì bỏ qua
    2. Gõ `npm start` để chạy app (admin và user thì dùng `npm run dev`)

> admin và user dùng cách tương tự để chạy
